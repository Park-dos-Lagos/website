# Administradores — Visão Geral
Este guia fornece um panorama completo das responsabilidades, fluxos e recursos que a equipe administrativa deve dominar para operar o site e o plugin VikBooking.

Público-alvo
- Administradores do site
- Operadores/reservas
- Gestores (relatórios e conciliação)

Objetivos deste documento
- Explicar responsabilidades diárias, semanais e mensais.
- Apontar guias operacionais passo-a-passo (VikBooking).
- Reunir procedimentos críticos: backups, manutenção, segurança e LGPD.

Principais responsabilidades (resumo)
- Manter disponibilidade e tarifas atualizadas no VikBooking.
- Monitorar e conciliar pagamentos (capturas, estornos, reembolsos).
- Gerenciar reservas: confirmação, alteração, cancelamento e registro de pagamentos manuais.
- Emissão de documentos: faturas/recibos e relatórios financeiros.
- Zelar por segurança, backups regulares e conformidade com LGPD.

Como usar esta área de documentação
- Use o menu lateral para navegar pelos tópicos administrativos.
- Para operação prática do VikBooking, consulte a pasta `docs/admin/vikbooking/` que contém guias por funcionalidade (configuração inicial, tarifas, reservas, pagamentos, etc.).
- Em caso de dúvidas operacionais, siga primeiro o checklist de "primeiros passos" e o guia passo-a-passo de operação (link abaixo).

Links rápidos
- Guia operacional (passo-a-passo): `docs/admin/vikbooking/operacao-step-by-step.md`
- Backup e manutenção: `docs/admin/vikbooking/backup-e-manutencao.md`
- Configuração inicial do VikBooking: `docs/admin/vikbooking/configuracao-inicial.md`

Contato e escalonamento
- Para problemas técnicos: equipe de TI / desenvolvedores (incluir contatos internos).
- Para questões financeiras: setor financeiro / contabilidade.

Nota sobre segurança e LGPD
- Trate dados de hóspedes com restrição de acesso.
- Ao compartilhar informações (planilhas, logs), aplique mascaramento quando necessário.
- Mantenha registro de operações (logs) e autorização para acessos administrativos.
